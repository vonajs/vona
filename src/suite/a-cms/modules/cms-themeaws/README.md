# vona-module-cms-themeaws

Cabloy-CMS Theme

## Installation

```bash
$ npm i vona-module-cms-themeaws
```

## Usage

Changing languange's theme to `cms-themeaws`

```javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cms-themeaws"
}
```
