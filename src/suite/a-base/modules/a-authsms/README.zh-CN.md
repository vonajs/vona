## 关于 a-authsms

CabloyJS 内置了短信认证模块`a-authsms`，可以实现`手机短信认证`的功能。主要有以下特性：

1. 提炼出通用的`手机短信验证`机制，可以通过`提供者`的方式快速对接各类短信服务提供商
2. 通过`手机短信验证`实现`用户注册`、`用户登录`、`用户认证`等功能
3. 内置`短信服务提供者`
   - 测试版本：不需要开通实际的短信服务，即可进行开发调试
   - 阿里云：只需填写参数即可快速对接部署阿里云短信服务
   - 持续增加对第三方服务的支持
