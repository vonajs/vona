# About

The First Blog Theme

## Installation

```bash
$ npm i cabloy-module-api-cms-themeblog
```

## Usage

Changing languange's theme to `cms-themeblog`

```javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cms-themeblog"
}
```
