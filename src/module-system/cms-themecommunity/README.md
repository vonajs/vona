## About

The First Community Theme

## Installation

```bash
$ npm i cabloy-module-api-cms-themecommunity
```

## Usage

Changing languange's theme to `cms-themecommunity`

```javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cms-themecommunity"
}
```
