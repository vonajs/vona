# Project Templates

To accelerate fullstack project development, Cabloy offers two project templates:

| Name                                            | UI                    | Description                                             | Demo                                                                       |
| ----------------------------------------------- | --------------------- | ------------------------------------------------------- | -------------------------------------------------------------------------- |
| [cabloy-basic](../cabloy-basic/introduction.md) | Daisyui + Tailwindcss | Provides out-of-the-box `Admin-Dashboard`               |                                                                            |
| [cabloy-start](../cabloy-start/introduction.md) | VuetifyJS             | Provides out-of-the-box `Website` and `Admin-Dashboard` | [Website](https://cabloy.com), [Admin-Dashboard](https://cabloy.com/admin) |

## Template Comparison

|                                                                 | cabloy-basic                                                                                  | cabloy-start                                            |
| --------------------------------------------------------------- | --------------------------------------------------------------------------------------------- | ------------------------------------------------------- |
| UI Library                                                      | Daisyui + Tailwindcss                                                                         | VuetifyJS                                               |
| Fullstack Capability                                            | Provides out-of-the-box `Admin-Dashboard`. You can develop `Website` capabilities on your own | Provides out-of-the-box `Website` and `Admin-Dashboard` |
| Pioneering Dual-Layer Tabs Navigation UI/UX                     | ✅                                                                                            | ✅                                                      |
| Dynamic Rendering for Form/Table                                | Provides basic UI components                                                                  | Provides a large number of UI components                |
| Admin-Dashboard SSR Support                                     | ✅                                                                                            | ✅                                                      |
| New CRUD Development Paradigm                                   | ✅                                                                                            | ✅                                                      |
| Best Practices of `Tanstack Table/Tanstack Form/Tanstack Query` | ✅                                                                                            | ✅                                                      |
