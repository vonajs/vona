# CabloyJS

## VonaJS和CabloyJS的关系

VonaJS 作为底层框架，提供可以实现能力闭环的模块集合。也就是说，基于 VonaJS 就可以开发一个完整的项目

为了进一步提升开发体验和开发效率，通过 CabloyJS 提供更多高级的模块集合
